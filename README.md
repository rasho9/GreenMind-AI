# GreenMind AI 🌱

**GreenMind AI** is an intelligent gardening assistant that helps users monitor plant health, receive AI-powered recommendations, manage gardening tasks, and make smarter gardening decisions through a simple and modern web application.

**Live Demo:** [https://green-mind-ai-lime.vercel.app/](https://green-mind-ai-lime.vercel.app/)

**GitHub Repository:** [https://github.com/rasho9/GreenMind-AI](https://github.com/rasho9/GreenMind-AI)

# 💡 Inspiration

Many people enjoy gardening but often struggle with plant care, disease identification, watering schedules, and choosing the right plants. We wanted to create an easy-to-use AI assistant that helps users make better gardening decisions without needing expert knowledge.

---

# ⚙️ How It Works

GreenMind AI brings together several smart tools in one application.

* Weather insights help users understand current conditions.
* AI Chat answers gardening questions.
* Plant Doctor assists with plant health guidance.
* Plant Library provides information about different plants.
* Recommendations offer helpful gardening suggestions.
* Tasks and Analytics help users organize and monitor their garden.

# 🛠 Installation

Clone the repository

```bash
git clone https://github.com/rasho9/GreenMind-AI.git
```

Go to the project folder

```bash
cd GreenMind-AI
```

Install dependencies

```bash
npm install
```

Run locally

```bash
npm run dev
```

Build project

```bash
npm run build
```

# 📦 Environment Variables

Create a `.env.local` file and add the required variables.

```env
VITE_ENABLE_LIVE_SERVICES=true

VITE_MAPTILER_API_KEY=YOUR_MAPTILER_KEY

OPENAI_API_KEY=YOUR_OPENAI_KEY
OPENWEATHER_API_KEY=YOUR_OPENWEATHER_KEY
PERENUAL_API_KEY=YOUR_PERENUAL_KEY
MAPTILER_API_KEY=YOUR_MAPTILER_KEY
```
